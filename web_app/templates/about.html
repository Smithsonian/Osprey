Osprey — DPO Collections Digitization Dashboard

Quality control and progress monitoring for Smithsonian digitization projects

Osprey checks images produced in mass digitization and imaging projects, displays results in a web dashboard, and helps vendors, project managers, and museum units monitor progress and catch problems early. It supports Mass Digitization, Imaging Services, and Informatics workflows across the Collections Digitization program.

Important: Osprey is not a permanent system of record. It provides a convenient way to review digitization products while projects are active. Do not link to specific pages or images—they may be removed without notice.

Digitization Program Office

Osprey is an open-source application developed by the Digitization Program Office. Source code is on GitHub (link is external).
